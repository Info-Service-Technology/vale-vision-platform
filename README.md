# Vale Vision Platform

Monorepo base para a plataforma de monitoramento visual de caçambas da Vale S.A.

## Estrutura

- `frontend/`: React + Vite + Material UI
- `backend/`: API FastAPI + SQLAlchemy + MySQL
- `inference/`: pipeline de visão computacional
- `infra/`: Terraform para AWS
- `docs/`: documentação técnica e de operação
